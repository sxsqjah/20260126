# Crypto-K Quants 官网

Crypto-K Quants 官方网站，支持中韩英三语。

## 项目简介

这是 Crypto-K Quants（面向加密货币市场的 K 系列量化团队）的官方网站，基于 Vue 3 + TypeScript + Vite 构建。

## 技术栈

- **框架**: Vue 3 + TypeScript
- **构建工具**: Vite
- **UI 框架**: Tailwind CSS
- **国际化**: Vue I18n
- **路由**: Vue Router
- **状态管理**: Pinia

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 项目结构

```
webwangye/
├── public/          # 静态资源
├── src/
│   ├── assets/      # 样式文件
│   ├── components/  # Vue 组件
│   ├── composables/ # 组合式函数
│   ├── locales/     # 国际化翻译文件
│   ├── router/      # 路由配置
│   ├── stores/      # Pinia 状态管理
│   ├── views/       # 页面视图
│   └── main.ts      # 应用入口
├── index.html
├── package.json
├── tailwind.config.js
└── vite.config.ts
```
